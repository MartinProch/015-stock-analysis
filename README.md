# Stock Analysis — Wavefront Lite

Shadcn-style stock dashboard built from the `014-wavefront` workflow.

## Features

- Watchlist with sortable tickers
- Elliott wave count, confidence score, targets and invalidation level
- Fibonacci retracements/extensions with range fallback
- Support/resistance, buy zones, volume bars and RSI divergence
- Added Wavefront overlays: WMA200, volume profile, regression channel, measured move, Fibonacci time zones and candlestick pattern labels
- RS Rank panel comparing watchlist momentum against SPY
- Portfolio panel for simple position tracking
- Keyboard shortcuts: `1-9`, arrows, `W/F/R/Z/V`, `P/C/M/T/K`, `S`, `?`

## Run

```bash
npm start
```

Open `http://localhost:3460/`.
